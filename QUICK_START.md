# Quick Start Guide

## TL;DR - Get Running in 2 Minutes

```bash
# 1. Install dependencies
pnpm install

# 2. Start development server
pnpm dev

# 3. Open http://localhost:3000
```

## What You'll See

### Profile Page
- Edit your profile (name, bio, title, institution, etc.)
- Add bento grid items (links, images, text, GitHub repos)
- Customize your layout with drag-and-drop

### Bento Grid
- Add different types of cards (Link, Image, Text, Repository, People, etc.)
- Drag and resize cards to customize layout
- Edit and delete cards

## Commands Explained

| Command | Purpose |
|---------|---------|
| `pnpm dev` | Start development server |
| `pnpm build` | Build for production |
| `pnpm start` | Run production server |
| `pnpm lint` | Run ESLint |
